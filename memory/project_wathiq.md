Built a full premium corporate website for a Syrian digital identity company branded as **WATHIQ** (وثيق).

**Why:** User requested a premium marketing website for meetings with banks, telecoms, government, and investors.

**Stack:** Next.js 14, React 18, Tailwind CSS, Framer Motion. Requires Node 18 (`.nvmrc` set).

**Pages:** Home, Solutions, Industries, Security, Partnership, Contact.

**How to apply:** Run `source ~/.nvm/nvm.sh && nvm use 18 && npm run dev` from `/Users/adhamkaraker/Desktop/New-project`. Build is clean and static-optimized.
